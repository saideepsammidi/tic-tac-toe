# Tic Tac Toe
#created by Saideep Sammidi

A browser-based Tic Tac Toe game built with vanilla HTML, CSS, and JavaScript — no dependencies.

## Features

- 2-player local gameplay (X vs O)
- Score tracking across rounds (wins and draws)
- Winning cells highlighted on game end
- Restart button to reset the board without clearing scores

## How to Play

Open `index.html` in any browser. Players alternate clicking cells to place their mark. First to get three in a row wins. Hit **Restart** to play again.

## Tech

Single `index.html` file — no build step, no frameworks.
